# برق نيوز v13

## أهم التعديلات
- إصلاح خطأ `normalize_article is not defined`.
- إضافة آخر 15 دقيقة وآخر 30 دقيقة.
- جلب عدد أكبر من الأخبار ودعم `next_url`.
- بحث السهم يطلب أخبار الرمز مباشرة.
- تحليل النظام وAI لكل خبر مع الآراء النصية.
- حفظ سعر لحظة الرصد عند توفر Snapshot.
- سجل تعلم يحفظ آخر وأعلى وأدنى حركة بعد الخبر.
- تعليق تطوير يوضح هل السوق أكد الإشارة أو خالفها.
- تحديث الأسعار تلقائيًا كل 30 دقيقة.
- تشغيل عامل الرصد تلقائيًا داخل Streamlit.
- إشعار نبض تيليجرام كل 30 دقيقة.
- كل الأوقات بتوقيت السعودية بصيغة `Sep 28th - 03:37am`.

## ملاحظة السعر
يتطلب السعر اللحظي/المتأخر أن تكون خدمة Stocks Snapshot متاحة في اشتراك Massive.
إذا لم تكن متاحة، يبقى الخبر والتقييم محفوظين ويظهر أن السعر غير متاح.

## ملاحظة Streamlit Cloud
ملفات JSON المحلية قد تُفقد عند إعادة تشغيل أو إعادة نشر التطبيق.
للاستخدام الدائم لاحقًا يفضّل نقل السجل إلى قاعدة خارجية.
